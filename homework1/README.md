## Requirements
Python 3.12.3
pytest 9.0.2

## How To Execute Tests

### Step 1: From the cs4300 repository navigate into homework1 directory

```
cd homework1/
```


### Step 2: Execute tests by running pytest, configuration is pre-defined by the pyproject.toml file.

```
pytest
```

## How To Execute Code

### Step 1: From the cs4300 repository navigate into homework1/src directory
```
cd homework1/src/
```

### Step 2: Execute tasks 1 to 7 using python 3

``` 
python3 task1.py
python3 task2.py 
python3 task3.py 
python3 task4.py 
python3 task5.py 
python3 task6.py 
python3 task7.py 
```
